# tasky

Go/Gin MongoDB todo application compatible with the selected source pattern.

Upstream source: https://github.com/dogukanozdemir/golang-todo-mongodb

## Runtime

```text
Port: 8080
MongoDB env var: MONGODB_URL
JWT env var: SECRET_KEY
Database: go-mongodb
```

## Local run

```bash
go mod tidy
go run main.go
```
